# DNA Sequence Analysis and Machine Learning

## Overview

This project performs DNA sequence analysis using BioPython and machine learning techniques.

The workflow includes reading DNA sequences from FASTA files, extracting biological features, training classification models, and comparing their performance.

## Technologies Used

* Python
* BioPython
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib

## Features Extracted

For each DNA sequence:

* Sequence Length
* GC Content
* Adenine (A) Frequency
* Thymine (T) Frequency
* Guanine (G) Frequency
* Cytosine (C) Frequency

## Dataset

DNA sequences are stored in FASTA format and processed using BioPython.

Example:

> seq1
> ATGCGTAGCTAGCTAGCTA

## Machine Learning Models

### K-Nearest Neighbors (KNN)

Used for DNA sequence classification based on extracted features.

### Gaussian Naive Bayes

Probabilistic classification model for sequence prediction.

## Model Evaluation

* Accuracy Score
* Confusion Matrix
* Classification Report
* Model Comparison

## Results

The project compares multiple machine learning algorithms and identifies the best-performing model based on prediction accuracy.
